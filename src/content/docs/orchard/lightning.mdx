Your mint settles real value over Lightning, so the Lightning node is the layer between
ecash and bitcoin on the wire. Orchard surfaces that node so you can see, at a glance,
whether your mint can actually pay and get paid.

## Dashboard

A dedicated Lightning dashboard is coming soon. In the meantime, the
[Home view](/orchard/home/) already summarises the node: an **Info** card with the
node alias, online status, connected peers, and open channels, and a **Channel Summary**
that totals your capacity and splits it into local (what you can send) and remote (what you
can receive).

<Aside type="note" title="Connecting your node">
  Which Lightning node Orchard talks to, and how, is set at install time with environment
  variables — see [Installation & Configuration](/install/configuration/). This page is
  about watching it once it's connected.
</Aside>
