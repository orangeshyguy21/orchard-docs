## Bitcoin Dashboard

Today, the [home page](/orchard/home/#bitcoin) contains information about your bitcoin node.
A dedicated dashboard is coming soon.

## Oracle

The Oracle is an optional feature that allows you to add in-app USD
conversions alonside Bitcoin values using only your own node's on-chain data.

You can backfill price history all the way back to 2020 in order to get acurate conversions 
on historical transactions. To learn more about the Oracle and how to enable it, see [Settings → Application → Bitcoin](/orchard/settings/#bitcoin).

## AI

Connecting a Bitcoin node also gives the [AI agent](/orchard/settings/#ai) read-only tools for knowing
what's happening on your node:

- **Network Info** — peer connections and network status.
- **Blockchain Info** — sync progress and chain state.
- **Bitcoin Analytics** — on-chain payment activity and fees over time.
