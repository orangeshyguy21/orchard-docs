Orchard aims to be useful out of the box and configurable when you need more. This
page covers the settings you'll reach for most — the public face of your mint — and
the optional features you can switch on as you grow.

## Mint information

The metadata wallets read when they connect to your mint — its public face.
Typically:

- **Name and description** — how your mint introduces itself.
- **Contact details** — how users reach you.
- **Icon and message of the day** — the finishing touches a wallet can display.

Keep it accurate: this is what a wallet shows someone before they trust your mint.

## Optional features

Orchard is meant to start simple and grow with you. Optional features stay off
until you turn them on, so the dashboard isn't cluttered with what you don't use.

<Aside type="caution" title="The feature list is still growing">
  Orchard is under active development, and which features are optional — and how you
  configure each one — is still settling, so the specifics land here as they
  stabilize. The [Orchard repository](https://github.com/cashubtc/orchard) has the
  source code in the meantime.
</Aside>
