Start here. Orchard is a self-hosted web app you run on a box you control — get it
installed and running first, then point it at each service from the pages that
follow. Once it's wired up, keys, balances, and mint health live in one dashboard,
so you tend the mint instead of babysitting a terminal.

## What you'll set up

- **Somewhere to run Orchard** — a VPS, home server, or any box you keep online.
- **The Orchard app itself**, installed and running on that machine.
- **Access to the dashboard**, ready to connect to your services.

<Aside type="tip" title="The groundskeeper">
  Orchard watches the mint so you don't have to refresh a log all day — health and
  activity surface where you already are.
</Aside>

<Aside type="caution" title="Installation steps are on the way">
  Orchard is under active development, and the step-by-step install guide is still
  being written — it lands right here. The
  [Orchard repository](https://github.com/cashubtc/orchard) has the source code in
  the meantime.
</Aside>
