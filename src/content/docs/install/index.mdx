Get Orchard running, then point it at the services you operate. Orchard *manages*
a mint rather than replacing one, so this section is about installing the app and
wiring it to each piece — whether you just built the stack with
[New Mint](/new-mint/) or already run your own Bitcoin, Lightning, and mint.

## Steps

Install Orchard first, then work through each service you run. The layers build on
one another — the mint settles against Lightning, which rides on Bitcoin — so the
order below is the path of least surprise.
