Starting from nothing? This is the suggested path to a **fully sovereign mint** —
every layer run by you, on hardware you control. A Cashu mint stands on three
layers, each settling down to the one below it. You'll build them bottom-up, then
let Orchard tend the result.

<Steps>

1. **[Bitcoin node](/new-mint/bitcoin-node/)** — the base layer. Your own Bitcoin
   Core node, the source of truth your Lightning node trusts.

2. **[Lightning node](/new-mint/lightning-node/)** — settles ecash in and out.
   LND or Core Lightning, anchored to the Bitcoin node below it.

3. **[Cashu mint](/new-mint/mint/)** — issues and redeems ecash against that
   Lightning node.

4. **[Orchard](/new-mint/orchard/)** — hand it the keys and tend the mint from one
   place.

</Steps>

<Aside type="tip" title="Already running some of this?">
  If you already operate a Bitcoin node, Lightning node, or mint, you don't need
  to rebuild it. Jump to [Already Have a Mint](/existing-mint/) and point Orchard
  at what you've got.
</Aside>
